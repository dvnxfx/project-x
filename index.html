Usando elementos do HTML no seu código

Keyframes

Fazendo uso do elemento abaixo é possível fazer com que o círculo se mova de forma suave:

animation: mov-div 3000ms" infinite alternate; — importante para o movimento funcionar.

animation-timing-function: ease-in-out; — suaviza o movimento.
